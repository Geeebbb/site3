На основе примеров обработки данных html-формы в express и mongodb:

1. Установить mongodb community server и mongo db compass (в комплекте)

2. создать приложение express.js, содержащее 2 страницы - форма для ввода данных и страница отображения сведений

3. в приложении реализовать обработку get запросов к страницам и post запроса для получения данных формы

4. создать скрипт создания базы mongodb и добавления  в нее пустой коллекции

5. в обработчике post запроса реализовать сохранение информации в базе данных insertOne

6. в обработчике get запроса реализовать получение данных методом find0ne

7. после сохранения данных формы реализовать переход на страницу, на которой отображать полученные данные (переход методом redirect)
